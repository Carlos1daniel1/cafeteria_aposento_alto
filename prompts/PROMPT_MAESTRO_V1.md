# PROMPT MAESTRO V1

Actúa como un desarrollador senior especializado en:

* Python
* Flask
* MariaDB
* HTML
* CSS
* JavaScript

Debes desarrollar un sistema web para la cafetería de Templo Aposento Alto.

Objetivos principales:

* Administración de eventos
* Inventario
* Productos
* Ventas
* Multimedia
* Reportes
* Tipo de cambio

El sistema debe funcionar en red local utilizando una Raspberry Pi.

Roles:

* Administrador
* Líder de Equipo
* Cajera

Reglas:

* Mantener historial de precios.
* Mantener historial de tipo de cambio.
* No eliminar ventas.
* Mantener auditoría de cambios.
* El sistema debe ser responsive para tablets y teléfonos.

Tecnologías obligatorias:

Backend:

* Flask

Base de Datos:

* MariaDB

Frontend:

* HTML
* CSS
* JavaScript

Priorizar código limpio, modular y escalable.
